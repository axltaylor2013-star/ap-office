# Roblox Game Marketer

Market and promote Roblox games for maximum player acquisition and retention.

## What This Skill Does
- Launch campaigns for new Roblox games
- Create promotional content (videos, posts, thumbnails)
- Build community around games
- Optimize game discovery and SEO
- Track metrics and iterate marketing

## When To Use
- Game is ready for launch/beta
- Need to increase player count
- Want to build community engagement
- Launching updates or events
- Creating buzz for monetization

## Key Strategies

### Launch Campaign
1. **Pre-Launch Buzz**
   - Teaser content across platforms
   - Developer updates/behind-scenes
   - Beta testing with friends
   - Social media countdown

2. **Launch Day Push**  
   - Coordinated multi-platform posting
   - Livestream gameplay
   - Ask friends/family to play first
   - Share on relevant Discord/Reddit communities

3. **Post-Launch Growth**
   - Regular content updates
   - Community engagement
   - Player feedback integration
   - Seasonal events

### Content Creation
- **Gameplay Videos**: 30-60sec clips showing best features
- **Tutorial Content**: How to play, tips & tricks
- **Update Announcements**: New features, events, items
- **Community Highlights**: Player achievements, builds, moments

### Platform Strategy

**Roblox Platform:**
- Compelling game icon & thumbnail
- SEO-optimized title and description  
- Regular updates to stay in "Recently Updated"
- Engage with player comments

**Social Media:**
- **TikTok/Reels**: Quick gameplay highlights, funny moments
- **YouTube**: Longer gameplay, tutorials, dev logs
- **Twitter/X**: Updates, community engagement, dev thoughts
- **Discord**: Build dedicated community server

**Reddit Marketing:**
- r/roblox - share appropriately 
- r/gamedev - development insights
- Niche gaming subreddits
- Local/regional subreddits if relevant

### Metrics to Track
- Daily/Monthly Active Users
- Session time and retention
- Monetization conversion rates
- Social media engagement
- User acquisition sources

- 100+ concurrent players within week 1
- 1000+ total visits within month 1
