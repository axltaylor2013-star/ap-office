# Character Design Descriptions

> **Instructions**: Fill in character details for your specific cartoon. Each entry provides the visual description used in AI art prompts and voice notes for TTS casting.

## Template

```markdown
### [Character Name]
- **Role**: Protagonist / Antagonist / Supporting
- **Appearance**: Physical description (height, build, hair, eyes, skin tone)
- **Outfit**: Default clothing/costume description
- **Distinguishing Features**: Unique visual identifiers (scars, accessories, etc.)
- **Expression Default**: Typical facial expression/mood
- **Color Palette**: Primary colors associated with this character
- **Voice Notes**: Voice qualities for TTS (pitch, tone, accent, energy level)
- **Prompt Snippet**: Copy-paste ready description for image prompts
```

## Example

### Captain Brix
- **Role**: Protagonist
- **Appearance**: Tall, athletic build, short messy brown hair, green eyes, light skin
- **Outfit**: Red jacket with gold trim, dark pants, boots
- **Distinguishing Features**: Scar over left eyebrow, always wears fingerless gloves
- **Expression Default**: Confident smirk
- **Color Palette**: Red, gold, brown
- **Voice Notes**: Mid-range, confident, slight rasp, energetic delivery
- **Prompt Snippet**: "Captain Brix, tall athletic man with short messy brown hair, green eyes, red jacket with gold trim, scar over left eyebrow, fingerless gloves, confident smirk"
